## 🌱 시작하기

이 강의 과정은 AI 에이전트 개발의 기본을 다루는 11개의 강의로 구성되어 있습니다. 각 강의는 독립적인 주제를 다루므로 원하는 곳부터 시작하세요!

이 과정은 다국어를 지원합니다. 지원되는 언어는 [여기](../..)에서 확인할 수 있습니다.

생성형 AI 모델로 처음 개발하시는 분은 21개의 강의로 구성된 [Generative AI For Beginners](https://aka.ms/genai-beginners) 과정을 참고하세요.

이 저장소에 [별(🌟)을 눌러주시고](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) [포크](https://github.com/microsoft/ai-agents-for-beginners/fork)하여 코드를 실행해 보세요.

### 준비물

각 강의에는 코드 예제가 포함되어 있으며, code_samples 폴더에서 확인할 수 있습니다. [이 저장소를 포크](https://github.com/microsoft/ai-agents-for-beginners/fork)하여 본인만의 복사본을 만드세요.

이 연습에 사용된 코드 예제는 Azure AI Foundry와 GitHub Model Catalogs를 활용하여 언어 모델과 상호작용합니다:

- [Github Models](https://aka.ms/ai-agents-beginners/github-models) - 무료 / 제한적  
- [Azure AI Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure 계정 필요

또한 이 과정에서는 Microsoft의 다음 AI 에이전트 프레임워크와 서비스를 사용합니다:

- [Azure AI Agent Service](https://aka.ms/ai-agents-beginners/ai-agent-service)  
- [Semantic Kernel](https://aka.ms/ai-agents-beginners/semantic-kernel)  
- [AutoGen](https://aka.ms/ai-agents/autogen)

코드 실행에 관한 자세한 내용은 [Course Setup](./00-course-setup/README.md)을 참고하세요.
